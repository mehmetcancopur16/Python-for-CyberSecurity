# Step 7: Data Structures - Sets

A Set is a collection which is unordered, unchangeable (but you can add/remove items), and most importantly: **unindexed and allows NO duplicate members.**



[Image of Set Intersection Venn Diagram]


## CyberSec Use Case
* **Deduplication:** You extracted 10,000 IP addresses from an Apache access log, but many are the same IP attacking repeatedly. Converting that list to a Set instantly removes all duplicates.
* **Intersection:** Finding common open ports between two different servers.
