# Step 6: Data Structures - Dictionaries

Dictionaries are used to store data values in **key:value** pairs. They are incredibly important in cybersecurity because they perfectly map to **JSON** (JavaScript Object Notation), which is the standard format for API responses (like querying Shodan, VirusTotal, or parsing configuration files).



## Key Features
* **Unordered:** (As of Python 3.7 they keep insertion order, but logically you access them via keys, not index).
* **Mutable:** You can change, add, or remove items.
* **Keys must be unique:** You cannot have two identical keys in the same dictionary.
