# Step 1: Introduction & Environment Setup

Before we hack the planet, we need to set up our environment and understand the philosophy of Python. 

## 🛠️ Environment Setup
To write and run Python code, you need:
1. **Python Interpreter:** Download and install the latest version from [python.org](https://www.python.org/). Make sure to check the box **"Add Python to PATH"** during installation.
2. **IDE/Code Editor:** Visual Studio Code (VS Code) or PyCharm are highly recommended.

## 🪄 The Magic of Python
Python values readability and simplicity. It allows you to express complex concepts in fewer lines of code compared to languages like C or Java.
