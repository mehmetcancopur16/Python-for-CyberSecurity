# Step 4: Operators in Python

Operators are special symbols that perform operations on variables and values. In cybersecurity, you will use operators constantly to calculate timeouts, compare IP ranges, check authentication states, and manipulate bytes.

## Types of Operators
1.  **Arithmetic Operators:** `+`, `-`, `*`, `/`, `%` (Modulus), `**` (Exponentiation).
2.  **Comparison (Relational) Operators:** `==` (Equal), `!=` (Not equal), `>`, `<`, `>=`, `<=`.
3.  **Logical Operators:** `and`, `or`, `not`. (Crucial for complex conditions).
4.  **Bitwise Operators:** `&` (AND), `|` (OR), `^` (XOR), `~` (NOT), `<<`, `>>`. 
    * *CyberSec Note:* Bitwise XOR (`^`) is heavily used in basic cryptography and malware obfuscation!
