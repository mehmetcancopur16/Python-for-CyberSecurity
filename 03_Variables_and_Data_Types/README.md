# Step 3: Variables and Data Types

In Python, variables are created the moment you first assign a value to them. You don't need to declare their type beforehand (this is called *dynamic typing*).

However, knowing your Data Types is crucial in cybersecurity. For example, an IP address is usually a `String`, but a network port must be an `Integer`. Mixing them up will crash your exploit!

## Fundamental Data Types
1.  **Integer (`int`):** Whole numbers (e.g., Port numbers, byte counts).
2.  **Float (`float`):** Decimal numbers (e.g., Timeouts, probability scores).
3.  **String (`str`):** Text data, enclosed in quotes (e.g., IP addresses, URLs, passwords).
4.  **Boolean (`bool`):** True or False (e.g., `is_admin`, `is_connected`).
