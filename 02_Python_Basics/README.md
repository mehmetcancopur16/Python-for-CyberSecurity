# Step 2: Python Basics - Syntax & Clean Code

In many programming languages (like C++ or Java), curly braces `{}` are used to define blocks of code. Python is different. It uses **indentation** (spaces or tabs) to define scope. This forces you to write clean, readable code. 

As a cybersecurity professional, you will often read other people's code (malware analysis, code review). Writing and reading clean code is a superpower.

## Key Concepts
* **Indentation:** Usually 4 spaces. Missing an indentation will cause an `IndentationError`.
* **Comments:** Use `#` for single-line comments. Good comments explain *why* you did something, not *what* you did.
* **Docstrings:** Used to document functions and modules using triple quotes `"""`.
